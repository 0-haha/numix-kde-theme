### Installation
Install:
```
git clone https://github.com/varlesh/numix-kde-theme.git
cd numix-kde-theme
sudo make install
```
Remove:
```
git clone https://github.com/varlesh/numix-kde-theme.git
cd numix-kde-theme
sudo make uninstall
```
