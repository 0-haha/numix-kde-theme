# Numix KDE Theme
A modern flat theme with a combination of light and dark elements. [See here](https://github.com/numixproject/numix-gtk-theme) for the GTK version.

### Installation
Install:
```bash
git clone https://github.com/numixproject/numix-kde-theme.git
cd numix-kde-theme
make install
```

Remove:
```bash
git clone https://github.com/numixproject/numix-kde-theme.git
cd numix-kde-theme
make uninstall
```
